# Pizza-Sales-Analysis-SQL-

## Project Overview
This project focuses on analyzing a Pizza Sales dataset using SQL to extract meaningful business insights related to sales performance, customer ordering behavior, and revenue generation. The analysis was performed using various SQL concepts including JOINs, aggregate functions, subqueries, CTEs, and window functions.
The project helps in understanding sales trends, top-performing pizzas, category-wise performance, and customer ordering patterns through real-world business queries.

## Objectives
- Analyze overall pizza sales performance
- Identify top-selling and highest revenue-generating pizzas
- Understand customer ordering trends by hour and category
- Calculate revenue contribution and cumulative sales growth
- Perform category-wise and time-based sales analysis
